## Steps to set up the starter template 

1. Clone the project 

'''
git clone https://github.com/karraanvesh/ExpressTypescriptStarterProject.git <ProjectName>
'''

2. Move in to the folder structure 

'''
cd <ProjectName>
'''

3. Install npm dependencies 

'''
npm i
'''

4. Create a new .env file in to the root directory and add the `PORT` env variable

'''
echo PORT=3000 >> .env
'''

5. Start the express server

'''
npm run dev
'''
